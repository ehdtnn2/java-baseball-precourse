# 기능 요구사항
- 볼 넘버는 1부터 9까지 서로 다른 수로 이루어진 3자리의 수로 구성된다.
- 볼 넘버 생성규칙 예외 상황
  - 사용자 입력이 1~9가 아닌 숫자(문자)인 경우
  - 사용자 입력이 중복된 경우
  - 사용자 입력이 3자리가 아닌 경우 (미입력 포함)
- 수비자는 볼 넘버를 생성한다.
- 공격자는 수비자의 볼 넘버를 추측하여 자신의 볼 넘버를 생성한다.
- 수비자는 공격자의 볼 넘버를 판단하여 결과(힌트)를 알려준다.
- 스트라이크 : 같은 수가 같은 자리
- 볼 : 같은 수가 다른 자리
- 낫싱 : 같은 수가 전혀 없음
